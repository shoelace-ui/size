### Opaque Size Mixin

### Example
```stylus
size 100% 30px
  /*yields*/
    width 100%
    height 30px

size 10px
  /*yields*/
    width 10px
    height 10px
```
  Test by running: ``` npm install ``` then ``` cd test && stylus < example.styl > example.css && open example.css```
